PROJECT SCOPE & COMPONENTS:
════════════════════════════════════════════════════════════════════════════════

1. STREET LIGHTING SYSTEM:
   - LED Street Light installation with 500 fixtures
   - Energy-efficient 120W LED luminaires
   - Estimated Value: Rs. 42,50,000

2. SOLAR INFRASTRUCTURE:
   - Installation of 200 Nos. 250W solar panels
   - Grid integration and battery backup system
   - Estimated Value: Rs. 24,00,000

3. ELECTRICAL DISTRIBUTION NETWORK:
   - Distribution panels, transformers, and circuit breakers
   - Underground cable network for power distribution
   - Estimated Value: Rs. 22,50,000

4. UNDERGROUND CABLING:
   - 4-core underground cables, 5000 meters
   - Termination, jointing, and testing
   - Estimated Value: Rs. 14,00,000

5. SCADA SMART MONITORING SYSTEM:
   - Real-time monitoring and control of all systems
   - Cloud-based dashboard and mobile app integration
   - Estimated Value: Rs. 18,75,000

6. PROJECT CONTINGENCY (5%):
   - Buffer for unforeseen costs and variations
   - Estimated Value: Rs. 6,08,750
